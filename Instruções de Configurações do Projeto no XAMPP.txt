Olá, segue os procedimentos para configurar o projeto:


Precisa ter o xampp instalado na máquina
Precisa ter o node.js instalado na máquina

Abra o Prompt de Comando na pasta do projeto (no htdocs) e digite:
npm init

Logo depois digite:
npm init --yes

Instale o AngularJS e Bootstrap a seguir:
npm install angular bootstrap

Instale o angular resource:
npm install angular-resource


Instale o angular route:
npm install angular-route

Instale o Jquery:
npm install jquery

Por último, instale o http-server:
npm install http-server -g
npm install http-server
